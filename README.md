# 📸 AI Personal Gallery Organizer

An AI-powered Personal Gallery Organizer that automatically detects, recognizes, and organizes photos based on faces using **TensorFlow**, **FaceNet**, **OpenCV**, **MTCNN**, and **Streamlit**.

---

## 🌟 Features

* 👤 Face Detection using MTCNN
* 🧠 Face Recognition using FaceNet Embeddings
* 📂 Automatic Gallery Organization
* 🔍 Search Photos by Person
* 📊 Gallery Analytics Dashboard
* 📤 Upload Multiple Images
* 🖥️ Interactive Streamlit Web Application

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Personal-Gallery-Organizer.git
```

### Move into the project

```bash
cd AI-Personal-Gallery-Organizer
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run src/app.py
```
